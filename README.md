# yinla-bot-website

This is the website for the YINLA Discord Bot.

## Development
To run the website locally, you can use the following command:

```bash
bun run dev
```

This will start the development server and you can access the website at `http://localhost:3000`.
## Build
To build the website for production, you can use the following command:
```bash
bun run build
```
This will create a production-ready build of the website in the `out` directory. You can then deploy the contents of the `out` directory to your web server.
